# 🌐 Lets

> **Empoderando la comunicación a través del rastreo ocular.**

Bienvenido a la documentación oficial de **Lets Vision**, una plataforma integral diseñada para romper las barreras de comunicación mediante el uso de inteligencia artificial y gestos oculares.

---

## 🚀 ¿Qué es Lets Vision?

Lets Vision no es solo una aplicación, es un ecosistema completo que combina visión artificial de vanguardia con interfaces de usuario accesibles. El proyecto se divide en dos pilares fundamentales:

### 👁️ Proyecto Karmi (Engine)
El "motor" de inteligencia artificial que procesa el video en tiempo real. Utiliza redes neuronales (TensorFlow/ONNX) y MediaPipe para detectar parpadeos y guiños con una precisión quirúrgica.
- [Explorar el motor Karmi](karmi.md)
- [Guía para integrar el Tracker en tu App](karmi_integration.md)

### 💻 LetsUI (Frontend)
El entorno de aplicaciones interactivas. Desde asistentes de voz hasta herramientas de escritura, LetsUI proporciona el marco necesario para que estas aplicaciones reaccionen a tus ojos.
- [Módulo Speaker (Voz)](ui_speaker.md)
- [Cómo crear tus propias Apps](main.md)

## 📢 Filosofía del Proyecto

Este proyecto nació con el objetivo de ser **local, rápido y privado**. 
- 🔒 **Privacidad**: Todo el procesamiento de video ocurre en tu PC, nada se envía a la nube.
- ⚡ **Velocidad**: Modelos optimizados para correr en hardware doméstico.
- 🤝 **Comunidad**: Diseñado para ser escalable; cualquiera puede crear y compartir sus propios paquetes de aplicaciones.
